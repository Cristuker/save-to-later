# Save to read later

A bot to save your favorites posts at bluesky

## Run locally

1. Clone repo
2. Npm install
3. npm run dev

## Libs

* redis
* @atproto/api
* nodemon
* TypeScript
* dotenv
* pm2
* node-cron
